# Installation

How to install dependencies and structure your app.

## New project

Use the `create-next-app` to create a new project.

```bash
npx create-next-app@latest my-app --typescript --tailwind --eslint
```

## Add Tailwind CSS

Install Tailwind CSS. Follow the [Tailwind CSS installation guide](https://tailwindcss.com/docs/guides/nextjs).

## Add dependencies

Add the following dependencies to your project:

```bash
pnpm add @radix-ui/react-slot
pnpm add class-variance-authority
pnpm add clsx tailwind-merge
pnpm add lucide-react
```

## Configure path mapping

I use the `@` alias to make it easier to import files. Add the following to your `tsconfig.json` file:

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

## Configure cn helper

I use a `cn` helper to make it easier to conditionally add Tailwind CSS classes. Here's how I define it:

```ts
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## Configure components.json

Running the following command will create a `components.json` file in your project:

```bash
npx shadcn-ui@latest init
```

You'll be asked a few questions to configure `components.json`:

```txt
Would you like to use TypeScript (recommended)? no / yes
Which style would you like to use? › Default
Which color would you like to use as base color? › Slate
Where is your global CSS file? › › app/globals.css
Do you want to use CSS variables for colors? › no / yes
Where is your tailwind.config.js located? › tailwind.config.js
Configure the import alias for components? › components
Configure the import alias for utils? › @/lib/utils
Are you using React Server Components? › no / yes (yes)
```

That's it! You can now start adding components to your project.
